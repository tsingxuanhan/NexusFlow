# 维度⑤分析: 环境健康

- **指标**: AIR_90 (环境空气污染归因DALYs, 年龄标准化, /10万人口)
- **数据年份**: 2021年截面数据（多源估计）

## 🔍 关键发现

1. **印度和中国的空气污染负担最重**: 与工业化、人口密度和能源结构有关
2. **巴西和俄罗斯相对较低**: 受益于较低的人口密度和不同的产业结构
3. **南非**: 煤炭依赖导致空气污染负担较高
