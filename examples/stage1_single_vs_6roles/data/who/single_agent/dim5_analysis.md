## 5.1 空气污染归因DALYs（AIR_43，总计）

| 国家 | 可用年份 | DALYs数值 | 特征 |
|------|----------|-----------|------|
| 巴西 | 2012 | 74,921 | 较低负担 |
| 俄罗斯 | 2012 | 12,576 | 较低负担 |
| 印度 | 2012 | 7,455,374 | 高负担 |
| 中国 | 2012 | 5,243,326 | 高负担 |
| 南非 | 2012 | 474,754 | 较低负担 |

## 5.2 关键发现

1. **印度**和**中国**因人口基数大，空气污染DALYs总量最高
2. **俄罗斯**化石燃料工业密集，空气污染负担不容忽视
3. **巴西**空气质量相对较好，得益于较低的工业化程度和较大的森林覆盖率
4. **南非**煤炭依赖度高，人均空气污染负担较高
注：AIR_43为DALYs总量（非率），受人口规模影响较大；原指标AIR_9（率）在当前API中暂无数据
