# 维度⑦：回测验证

- **评估指标**: MAPE（平均绝对百分比误差）

## 7.3 总体MAPE: 2.53%

## 7.4 分析

整体MAPE为2.53%，线性趋势模型预测精度**良好**，说明BRICS国家预期寿命变化总体呈线性趋势。

各国差异分析：
- **巴西**（APE=0.9%）：预测准确，线性趋势良好
- **俄罗斯**（APE=0.1%）：预测准确，线性趋势良好
- **印度**（APE=0.6%）：预测准确，线性趋势良好
- **中国**（APE=1.4%）：预测准确，线性趋势良好
- **南非**（APE=9.8%）：偏差较大，主要受HIV/AIDS epidemic导致的预期寿命先降后升的非线性模式影响
