## 🧩 Coder 核心计算逻辑

```python
# ===== 线性回归 (OLS) =====
def linear_regression(x, y):
    n = len(x)
    sum_x, sum_y = sum(x), sum(y)
    sum_xy = sum(xi*yi for xi,yi in zip(x,y))
    sum_x2 = sum(xi**2 for xi in x)
    denom = n * sum_x2 - sum_x**2
    slope = (n*sum_xy - sum_x*sum_y) / denom
    intercept = (sum_y - slope*sum_x) / n
    y_mean = sum_y / n
    ss_tot = sum((yi-y_mean)**2 for yi in y)
    ss_res = sum((yi-(slope*xi+intercept))**2 for xi,yi in zip(x,y))
    r_squared = 1 - ss_res/ss_tot
    return slope, intercept, r_squared

# ===== 舒适度指数 =====
def comfort_index(tmax, tmin, prcp):
    tavg = (tmax + tmin) / 2
    temp_comfort = max(0, 100 - abs(tavg - 21) * 8)
    prcp_comfort = max(0, 100 - abs(prcp - 600) * 0.08)
    diurnal_comfort = max(0, 100 - abs((tmax-tmin) - 10) * 5)
    return temp_comfort*0.5 + prcp_comfort*0.3 + diurnal_comfort*0.2

# ===== MAPE =====
def mape(actual, predicted):
    return abs((actual - predicted) / actual) * 100
```

## 🎯 Synthesizer 统一叙事

> 华北五城市正在经历**同步的变暖变湿过程**。温度以每年+0.02~+0.08°C的速率上升（西安最快），降水以每年+3~+17mm的速率增加（北京最快）。2021年的极端降水事件可能标志着新的气候态。线性回测表明升温可能在加速。在有限的气候变量维度下，西安展现出最高的宜居性评分，但需注意这一排名未考虑空气质量等关键因素。
>
> **核心风险**: 极端高温频率可能增加（济南、西安已有显著表现），极端低温频率可能减少但冬季寒潮仍会发生。城市规划需适应"更热更湿"的未来。
