### **研究分析框架：基于NOAA+WHO数据的气候-健康关联分析**

#### **1. 研究假设体系**

基于现有科学证据（如IPCC报告、环境流行病学文献），我们提出以下三个核心假设，以检验气候因素对健康结局的直接与间接影响。

*   **H1（直接效应假设）：** 年平均温度与温度极值的升高，与呼吸系统疾病发病率和心血管疾病死亡率的增加呈显著正相关。 *(理论依据：高温热浪直接导致热应激、加重心肺负担；极端温度事件（如寒潮）诱发呼吸系统感染和心血管事件。)*
*   **H2（非线性与阈值效应假设）：** 气候因素对健康的影响并非线性，存在“J型”或“U型”关系。年降水量和降水变异系数的增加，在特定阈值（如洪涝灾害）下会显著加剧健康风险。 *(理论依据：流行病学常见暴露-反应关系；干旱与洪涝通过影响病原体传播、粮食安全、精神压力等间接影响健康。)*
*   **H3（效应修饰假设）：** 气候对健康的影响强度会因社会经济条件（GDP、城市化率）和医疗资源（医疗资源密度）的不同而呈现显著的区域异质性。低GDP、低医疗资源密度的地区，其居民对气候变化的脆弱性更高。 *(理论依据：社会脆弱性理论；环境正义视角下，贫困地区适应能力差，健康风险放大。)*

#### **2. 变量体系与操作化定义**

为确保分析的可重复性和有效性，所有变量需进行标准化定义。

| 变量类型 | 变量名称 | 操作化定义 | 数据来源 | 单位/处理说明 |
| :--- | :--- | :--- | :--- | :--- |
| **自变量** | 年平均温度 | 年度内日平均气温的算术平均值 | NOAA (GHCN, ERA5等) | 摄氏度 (°C) |
| | 温度极值 | 年最高气温的95%分位数 或 年最低气温的5%分位数 | NOAA | 摄氏度 (°C) ，可选择热浪指数 (HWDI) 或寒潮强度 |
| | 年降水量 | 年度内总降水量 | NOAA (GPCP, TRMM等) | 毫米 (mm) |
| | 降水变异系数 | `(年降水量标准差 / 年降水量均值) * 100%` | NOAA | 百分比 (%)，衡量降水年际波动性 |
| **因变量** | 呼吸系统疾病发病率 | 每10万人口中，当年新发的呼吸系统疾病 (ICD-10: J00-J99) 病例数 | WHO (全球卫生观察站) | 比率 (1/100,000) |
| | 心血管疾病死亡率 | 每10万人口中，当年因心血管疾病 (ICD-10: I00-I99) 死亡的人数 | WHO | 比率 (1/100,000) |
| **协变量** | GDP | 人均国内生产总值 (购买力平价调整) | 世界银行 | 美元 (USD)，对数转换以处理偏态分布 |
| | 城市化率 | 城镇人口占总人口的比例 | 世界银行 / 联合国 | 百分比 (%) |
| | 医疗资源密度 | 每千人口医生数 或 每千人口医院床位数 | WHO | 连续变量，需选择信度高的指标 |

#### **3. 统计方法选择与分析策略**

我们将采用分层递进的统计策略，从描述到推断，逐步验证假设。

**阶段一：数据预处理与描述性分析**
*   **方法：** 描述性统计 (均值、标准差、中位数、四分位数)；数据可视化 (时间序列图、散点图矩阵、箱线图)。
*   **目的：** 识别缺失值、异常值；检验变量分布形态；初步观察趋势。

**阶段二：相关性分析**
*   **方法：**
    *   **Pearson相关系数：** 用于评估连续变量间（如温度与发病率）的线性相关强度。
    *   **Spearman秩相关系数：** 用于评估变量间的单调关系（非线性关系），对异常值不敏感。特别适用于H2中可能存在的非线性关系。
*   **输出：** 相关系数矩阵，热力图。

**阶段三：核心效应检验 (验证H1 & H2)**
*   **方法1：多元线性回归 (MLR)**
    *   **模型：** `健康结局 ~ β1*温度 + β2*降水 + β3*协变量 + ε`
    *   **处理：** 引入自变量的二次项 (如 `温度²`) 以检验H2中的U型关系。使用方差膨胀因子(VIF)诊断多重共线性。
*   **方法2：分布滞后非线性模型 (DLNM)**
    *   **适用性：** 气候因素对健康的影响往往存在滞后（如热浪后数天心血管死亡增加）。DLNM能同时捕捉暴露-反应的非线性关系和滞后效应。
    *   **模型设定：** 以“年平均温度”或“温度极值”为暴露变量，选择自然立方样条函数定义暴露-反应和滞后-反应维度。滞后天数可设为0-30天。
    *   **输出：** 三维暴露-滞后-风险图；累计风险曲线。

**阶段四：效应修饰与异质性分析 (验证H3)**
*   **方法：分层分析 (Stratified Analysis)**
    *   **分组变量：** 根据GDP、城市化率、医疗资源密度的中位数或三分位数，将样本分为高/低组。
    *   **分析：** 在每个分层内重复阶段三的MLR或DLNM模型。
    *   **检验：** 使用似然比检验或Wald检验比较不同分层间回归系数是否存在统计学显著差异。也可采用交互项分析 (如 `温度 * GDP`)。
