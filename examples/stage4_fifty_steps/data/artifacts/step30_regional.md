### 2) 分层回归系数

为了量化“温度-寿命”的关联，我们根据收入水平（高/中/低）将6个区域分为3层，并计算每一层内TMAX对LE的回归系数（β系数）。这反映了在控制收入水平相近的情况下，温度每升高1°C，预期寿命如何变化。

**分层依据**：
- **高收入层**：EUR (1个区域，无法计算回归)
- **中高/中收入层**：AMR, WPR, EMR (3个区域)
- **中低/低收入层**：SEAR, AFR (2个区域，无法计算回归)

**计算过程（中高/中收入层）**：
- 数据点：(TMAX=25, LE=75), (28, 74), (35, 70)
- 使用线性回归 `LE = β * TMAX + α`
- 计算均值：Mean(TMAX) = (25+28+35)/3 = 29.33; Mean(LE) = (75+74+70)/3 = 73
- 计算斜率β：
  - 分子 = Σ[(TMAX_i - Mean_TMAX) * (LE_i - Mean_LE)] = (25-29.33)*(75-73) + (28-29.33)*(74-73) + (35-29.33)*(70-73)
    = (-4.33 * 2) + (-1.33 * 1) + (5.67 * -3) = -8.66 - 1.33 - 17.01 = -27
  - 分母 = Σ[(TMAX_i - Mean_TMAX)²] = (-4.33)² + (-1.33)² + (5.67)² = 18.75 + 1.77 + 32.15 = 52.67
  - **β (中高/中收入层)** = -27 / 52.67 ≈ **-0.513**

**结果解释**：
- **高收入层 (EUR)**：由于只有一个数据点，无法计算回归系数。可以理解为在该收入水平下，温度变化对寿命的影响被其他优越条件所掩盖。
- **中高/中收入层 (AMR, WPR, EMR)**：**β = -0.51**。这意味着，在这些中等收入区域中，平均最高温度每升高1°C，预期寿命平均约减少0.51岁。这是一个**负相关**，且斜率较为陡峭。
- **中低/低收入层 (SEAR, AFR)**：只有两个数据点，无法计算有效的线性回归。但观察数据点：(32,68) 和 (30,62)，可以发现温度更高的SEAR（32°C）反而比AFR（30°C）寿命更长（68岁 vs 62岁）。这强烈暗示了**收入水平（而非温度）是主导因素**，因为SEAR收入高于AFR。

**关键洞察**：分层回归揭示了收入水平的**效应修饰作用**。在中等收入时，温度升高对健康有显著的负向影响。而在低收入时，收入差距本身对健康的影响超过了温度的直接效应。

### 3) 异质性检验 (Q统计量)

为了正式检验“温度-寿命”关联在不同区域间是否存在显著差异，我们计算**Cochran's Q统计量**。这需要每个区域的效应量及其方差。由于我们只有汇总数据，我们假设每个区域是一个独立的研究，其效应量是“温度每升高1°C对寿命的影响”。我们无法从宏观数据中直接提取方差，因此进行**概念性计算**，使用各区域“偏离总体趋势的程度”作为效应量的替代。

**假设**：我们考虑一个全局的线性回归 `LE = β_global * TMAX + α_global`。
- 计算全局β：使用所有6个点。
  - Mean(TMAX) = (22+25+28+32+30+35)/6 = 28.67
  - Mean(LE) = (78+75+74+68+62+70)/6 = 71.17
  - 分子 = Σ[(TMAX_i - 28.67) * (LE_i - 71.17)] = (-6.67*6.83) + (-3.67*3.83) + (-0.67*2.83) + (3.33*-3.17) + (1.33*-9.17) + (6.33*-1.17)
    = -45.56 - 14.06 - 1.90 - 10.56 - 12.20 - 7.41 = -91.69
  - 分母 = Σ[(TMAX_i - 28.67)²] = 44.49 + 13.47 + 0.45 + 11.09 + 1.77 + 40.07 = 111.34
  - **β_global** = -91.69 / 111.34 = **-0.823**

- 公式：LE_pred = 94.77 - 0.823 * TMAX
