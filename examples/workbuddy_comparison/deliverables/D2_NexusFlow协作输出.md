## CDoL信息流架构说明

本实验中，10个Agent按以下CDoL数据切片规则运作：

| Agent | 可访问数据子集 | 不可访问数据 |
|-------|----------------|--------------|
| **Coordinator** | 全局任务状态、各Agent完成状态、共识度历史 | 原始数据明细 |
| **Planner** | 任务目标、Researcher的数据质量报告 | 原始数据值 |
| **Researcher** | 全部15指标原始数据（数据获取角色） | 其他Agent的中间结论 |
| **Miner** | GDP增速、通胀率时间序列（挖掘模式） | 财政明细数据 |
| **Assayer** | Miner的发现 + Researcher的原始数据（交叉验证） | Planner的策略 |
| **Executor** | Planner的步骤 + Researcher清洗后的数据 | Miner的原始发现 |
| **Caster** | 代码执行环境 + Executor的分析需求 | 其他Agent的内部状态 |
| **Artisan** | 全部指标（领域知识审核需要全局视角） | 无 |
| **Reviewer** | 所有Agent的输出（质量审查需要全局视角） | 无 |
| **Archivist** | 最终结论 + 共识度记录 | 过程中间状态 |

### 子任务1：结构性转折点（修正后，20国各3个）

> 以下为10Agent协作后的修正结果，标注修正点

#### 美国（USA）
1. **1982-1983年**：沃尔克紧缩尾声。GDP 5年均值从0.16%→4.58%（shift=+4.42）。通胀13.5%→3.2%。驱动：高利率遏制通胀，开启长扩张。
2. **2008-2009年**：全球金融危机。GDP从1.9%→-2.6%，5年均值1.94%→0.64%。驱动：次贷危机→系统性金融崩溃。
3. **2020年**：COVID冲击。GDP -2.8%，和平时期最大跌幅。驱动：全球封锁供需双冲击。

#### 中国（CHN） *[Artisan修正：区分为两个独立事件]*
1. **1989年**：宏观紧缩。GDP从11.2%→4.2%，通胀18%。驱动：紧缩政策+外部环境。*[Artisan补充：这是"政策驱动型转折"]*
