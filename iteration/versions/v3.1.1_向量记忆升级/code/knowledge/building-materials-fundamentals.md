## 一、硅酸盐水泥矿物组成

### 1.1 四大熟料矿物 (Bogue Notation)

| 矿物 | 化学式 | 缩写 | OPC中含量 | 水化速率 | 强度贡献 | 水化热(J/g) |
|------|--------|------|-----------|----------|----------|-------------|
| 硅酸三钙 | 3CaO·SiO2 | C3S | 45-65% | 快速(1-28d) | 主导28d强度 | ~500 |
| 硅酸二钙 | 2CaO·SiO2 | C2S | 15-30% | 慢速(周-年) | 长期强度 | ~250 |
| 铝酸三钙 | 3CaO·Al2O3 | C3A | 6-12% | 极快 | 低(需控制) | ~860 |
| 铁铝酸四钙 | 4CaO·Al2O3·Fe2O3 | C4AF | 8-12% | 中速 | 中等 | ~420 |

- **来源**: Britannica cement article + Engineers Views concrete guide
- **验证状态**: ✅ 多源验证

### 1.2 硅酸三钙(C3S)水化

**反应方程式**:
```
2C3S + 6H → C3S2H3 + 3CH
```

**核心要点**:
- 水化产物: C-S-H凝胶 + 氢氧化钙(CH)
- 贡献约70%的28天强度
- 水化热较高，需控制大体积混凝土温升
- **来源**: PSU CE584 Hydration Notes + Wikipedia Concrete
- **验证状态**: ✅ 交叉验证

### 1.3 硅酸二钙(C2S)水化

**反应方程式**:
```
2C2S + 4H → C3S2H3 + CH
```

**核心要点**:
- 水化产物同C3S，但CH产量较低
- 负责28天后的强度增长
- 水化热仅为C3S的一半
- **来源**: PSU CE584 + Engineers Views
- **验证状态**: ✅ 交叉验证

### 1.4 铝酸三钙(C3A)水化与石膏控制

**无石膏时的反应**:
```
C3A + 6H → C3AH6 (水榴石)
```
⚠️ **闪凝风险**：无石膏时C3A快速水化导致闪凝

**有石膏时的反应**:
```
C3A + 3CSH2 + 26H → C6AS3H32 (钙矾石/Ettringite)
```
针状钙矾石晶体包裹C3A表面，阻止进一步快速水化

**石膏耗尽后**:
```
2C3A + C6AS3H32 + 22H → 3C4ASH18 (单硫型)
```
- 外部硫酸盐侵入可将单硫型转回钙矾石(体积膨胀227%)
- **硫酸盐侵蚀的内部机理**
- **来源**: PSU CE584 + Wikipedia Hydration
- **验证状态**: ✅ 交叉验证

## 二、水化产物

### 2.1 主要水化产物组成

**成熟水泥浆体的组成**（按体积）:

| 水化产物 | 体积分数 | 重要性 |
|----------|----------|--------|
| C-S-H凝胶 | 50-60% | **最关键，强度来源** |
| 氢氧化钙(CH) | 20-25% | 易溶，有害相 |
| 钙矾石/单硫型 | 15-20% | 需控制 |
| 毛细孔/气孔 | 5-6% | 致密性 |

- **来源**: PSU CE584 + CIMSA Turkey
- **验证状态**: ✅ 交叉验证

### 2.2 C-S-H凝胶

**结构特性**:
- 结晶度差的纤维状凝胶
- **非固定化学计量**，C/S比可变(1.5-2.0)
- 典型OPC中C/S ≈ 1.7-1.8
- **来源**: PSU CE584 + Civil Rack + Britannica
- **验证状态**: ✅ 交叉验证

### 2.3 氢氧化钙(CH)

- 化学式: Ca(OH)2
- 结晶良好的六方板状
- 占水泥浆体体积20-25%
- **可溶性高** → 碱性来源但也是薄弱相
- **来源**: PSU CE584 + Wikipedia
- **验证状态**: ✅ 交叉验证

## 三、配合比设计原理

### 3.1 水灰比(W/C)理论 - Abrams定律

**Abrams定律 (1918)**:
```
fc = A × (w/c)^(-B)
```

**水灰比-强度对应表**:

| w/c | 28d强度(MPa) | 混凝土等级 | 典型应用 |
|-----|--------------|-----------|---------|
| 0.30 | 50-60 | M40+ | 高性能混凝土 |
| 0.35 | 45-55 | M35-M40 | 桥梁、海洋结构 |
| 0.40 | 35-45 | M30-M35 | 工业地坪 |
| 0.45 | 30-40 | M25-M30 | 柱、梁、板 |
| 0.50 | 25-35 | M20-M25 | 一般RC结构 |
| 0.60 | 15-22 | M10-M15 | PCC、找平层 |

- **来源**: Wikipedia Water-Cement Ratio + The Civil Studies
- **验证状态**: ✅ 交叉验证
