## 主题索引

### 1. 基础知识 (Fundamentals)
- 硅酸盐水泥矿物组成与水化机理
- 水化产物（C-S-H/CH/AFt/AFm）形成与转化
- 混凝土配合比设计原理
- 建筑材料标准体系（GB/ASTM/EN）

### 2. 低碳水泥体系 (Low-Carbon Cements)
- **SSC** 超硫酸盐水泥（Super Sulfate Cement）
- **MBCMs** 镁基胶凝材料（Magnesium-Based Cementitious Materials）
- **LC3** 石灰石煅烧黏土水泥（Limestone Calcined Clay Cement）
- **CSA** 硫铝酸盐水泥
- **碱激发材料** 地聚物（Geopolymer）
- **多元耦合固废体系** 哈工大+四川诺优（2026金奖）

### 3. 纳米改性 (Nano-Modification)
- Nano SiO₂ 晶核效应与火山灰反应（40nm最优）
- Nano CaCO₃ 化学作用与填充效应
- ZIF-8 MOF 阻锈剂
- 纳米材料协同效应

### 4. 耐久性 (Durability)
- 氯离子侵蚀机理与防护
- 硫酸盐侵蚀机理与防护
- 碳化机理与控制
- **CO₂碳化养护技术**
- 冻融循环破坏
- 碱骨料反应（ASR）

### 5. 辅助胶凝材料 (SCMs)
- 粒化高炉矿渣（GGBS）
- 粉煤灰（Fly Ash）
- 硅灰（Silica Fume）
- 偏高岭土（Metakaolin）
- **固废陶粒骨料** 百万吨级低成本
- **赤泥基胶凝材料** 固废掺量90%+

### 6. AI+建材 (AI & Building Materials)
- 机器学习预测混凝土性能
- 深度学习水化预测
- 大语言模型材料发现
- **AI+材料发现平台**（海螺+华为、SmartMix Web3、Giatec Roxi）
- **DeepMind GNoME** 材料预测

### 7. CO₂捕集与封存 (CO₂ Capture)
- **MicroMg** 镁基生物杂化材料（CSIC 2026）
- 纳米孔隙碳化机理（Nature 2026）
- 碳化混凝土技术

### 8. 3D打印混凝土 (3DCP)
- 低碳3D打印材料
- 可打印性评价指标
- 废旧材料在3D打印中的应用
- 数字化设计与智能控制
