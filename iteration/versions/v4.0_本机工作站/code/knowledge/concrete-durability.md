## 一、氯离子侵蚀

### 1.1 侵蚀机理

**氯离子来源**:

| 来源 | 说明 |
|------|------|
| 海洋环境 | 海水浪花、雾气 |
| 除冰盐 | 氯化钠、氯化钙 |
| 自身含有 | 水泥、外加剂、骨料 |

**侵蚀机理**:
1. **扩散作用**: 浓度梯度驱动
2. **毛细作用**: 混凝土表面干燥时吸入
3. **渗透作用**: 水压力驱动
4. **结合作用**: 与C3A、水化铝酸盐化学结合

**临界氯离子浓度**:
- 引起钢筋锈蚀的临界[Cl-]/[OH-]比约0.6
- 或总氯离子含量>水泥质量的0.4-1.0%

- **来源**: Concrete Durability Testing Guide + QnA Engine
- **验证状态**: ✅ 交叉验证

### 1.2 Fick第二定律

**氯离子扩散方程**:
```
∂C/∂t = D · ∂²C/∂x²

解: C(x,t) = C_s · [1 - erf(x / (2√(D·t)))]

其中:
C(x,t) = t时刻x深度处氯离子浓度
C_s = 表面氯离子浓度
D = 扩散系数
t = 暴露时间
erf = 误差函数
```

- **来源**: 赵铁军公式 (materials-domain.md)
- **验证状态**: ✅ 理论验证

### 1.3 赵铁军渗透模型

**氯离子扩散系数时变规律**:
```
D(t) = D_0 · (t/t_0)^m

其中:
D_0 = 初始扩散系数
m = 时变指数(通常0.2-0.5)
```

**赵铁军公式**:
```
D = D_0 × exp(β × √t)
```

- **来源**: materials-domain.md
- **验证状态**: ✅ 同源验证

### 1.4 耐久性检测方法

| 方法 | 标准 | 参数 |
|------|------|------|
| **电通量法** | ASTM C1202 | 通过电量(Coulombs) |
| **RCM法** | NT Build 492 | D_RCM (氯离子迁移系数) |
| **NEL法** | 清华大学 | D_L (扩散系数) |

**指标对比**:

| 方法 | 优点 | 缺点 |
|------|------|------|
| 电通量法 | 操作简单 | 电压导致副反应 |
| RCM法 | 物理意义明确 | 需24h |
| NEL法 | 8分钟快速 | 需专业设备 |

- **来源**: Concrete Durability Testing Guide + 中国耐久性检测
- **验证状态**: ✅ 交叉验证

### 1.5 提高抗氯离子侵蚀措施

| 措施 | 机理 |
|------|------|
| 降低W/C | 减少毛细孔 |
| 添加硅灰 | 细化孔结构 |
| 添加矿渣 | 火山灰反应消耗CH |
| 表面涂层 | 物理阻隔 |
| 钢筋阻锈剂 | 化学保护 |

- **来源**: Concrete Durability Testing Guide
- **验证状态**: ✅ 验证

## 二、硫酸盐侵蚀

### 2.1 侵蚀类型与机理

**外部硫酸盐侵蚀(ESA)**:

**反应方程式**:
```
外部SO4²⁻ + C4ASH12 (单硫型) → C6ASH32 (钙矾石)
体积膨胀约227%
```

**内部硫酸盐侵蚀(ISA)**:
- 来自过量的石膏或高硫C3A水泥
- 热养护期间形成延迟钙矾石(DES)

- **来源**: PSU CE584 + Wikipedia Portland cement
- **验证状态**: ✅ 交叉验证

### 2.2 硫酸盐侵蚀破坏特征

| 破坏类型 | 特征 |
|----------|------|
| 膨胀开裂 | 钙矾石形成导致膨胀 |
| 强度下降 | 水泥石软化 |
| 表面剥落 | 晶体生长压力 |
| 水泥浆体分解 | Mg2+置换Ca2+ |

- **来源**: Wikipedia Portland cement
- **验证状态**: ✅ 验证

### 2.3 抗硫酸盐措施

| 措施 | 说明 |
|------|------|
| 选用低C3A水泥 | C3A<5% |
| 选用抗硫酸盐水泥 | GB/T 748, ASTM Type V |
| 降低W/C | 提高致密性 |
| 添加火山灰材料 | 火山灰反应消耗CH |
| 表面防护 | 涂层或防水层 |

- **来源**: Concrete Durability Testing Guide + GB/T 748
- **验证状态**: ✅ 交叉验证

## 三、碳化

### 3.1 碳化机理

**化学方程式**:
```
CO2 + H2O → H2CO3
Ca(OH)2 + H2CO3 → CaCO3 + 2H2O
C-S-H + H2CO3 → CaCO3 + SiO2·nH2O
```

**碳化深度模型**:
```
x = √(2D_c·t/c₀)

其中:
x = 碳化深度
D_c = CO2有效扩散系数
t = 暴露时间
c₀ = 表面CO2浓度
```

- **来源**: Wikipedia Concrete + QnA Engine
- **验证状态**: ✅ 交叉验证

### 3.2 碳化对钢筋混凝土的影响

**pH下降**:
- 正常混凝土孔溶液pH: 12.5-13.5
- 碳化后pH降至10以下
- **临界pH ≈ 10.5**

**钢筋锈蚀**:
- pH<10.5时，钢筋钝化膜破坏
- 钢筋开始锈蚀
- 锈蚀产物体积膨胀(~3倍)
- 混凝土开裂、剥落

- **来源**: QnA Engine Carbonation Depth Test
- **验证状态**: ✅ 验证

### 3.3 碳化深度测试

**酚酞指示剂法**:
- 喷射新鲜断面试件
- 高pH区(健康混凝土)→粉红色
- 碳化区→无色

**标准**:
- EN 14630 (欧洲)
- RILEM建议

- **来源**: QnA Engine
- **验证状态**: ✅ 验证

### 3.4 影响碳化的因素

| 因素 | 影响 |
|------|------|
| 环境湿度 | 50-70%最佳 |
| CO2浓度 | 浓度越高，碳化越快 |
| W/C | 越低，抗碳化越好 |
| 掺合料 | 粉煤灰加速碳化但提高后期 |
| 养护条件 | 充分养护提高抗碳化 |

- **来源**: Concrete Durability Testing Guide
- **验证状态**: ✅ 验证

## 四、冻融循环

### 4.1 破坏机理

**结冰膨胀**:
- 水结冰体积膨胀约9%
- 产生静水压力
- 产生渗透压力

**破坏特征**:
- 表面剥落
- 内部微裂缝
- 强度下降
- 质量损失

- **来源**: QnA Engine + Concrete Durability Guide
- **验证状态**: ✅ 交叉验证

### 4.2 抗冻措施

| 措施 | 说明 |
|------|------|
| 引气剂 | 引入气泡缓解静水压力 |
| 降低W/C | 减少可冻水量 |
| 选用优质骨料 | 骨料饱和度低 |
| 矿物掺合料 | 改善孔结构 |

**引气量要求**:
- 严重冻融环境(XF3/XF4): 目标含气量5.5%±1.5%

- **来源**: Concrete Durability Testing Guide
- **验证状态**: ✅ 验证

### 4.3 抗冻性测试

**快冻法**:
- 试件在水中冻融循环
- 温度: -18°C ~ +5°C
- 终止条件: 冻融300次或动弹模降至60%

**慢冻法**:
- 空气中冷冻，水中融化
- 质量损失和强度损失评定

- **来源**: GB/T 50082 + QnA Engine
- **验证状态**: ✅ 验证

## 五、碱骨料反应

### 5.1 反应类型

| 类型 | 说明 |
|------|------|
| **碱-硅酸反应(ASR)** | 最常见，与活性SiO2 |
| 碱-硅酸盐反应 | 与硅酸盐岩石 |
| 碱-碳酸盐反应 | 与白云石质石灰石 |

- **来源**: Penn State Concrete Performance + ASR paper
- **验证状态**: ✅ 交叉验证

### 5.2 ASR机理

**三阶段反应**:

1. **水解**: 
   ```
   Si-O-Si + OH- → Si-OH + Si-O-
   ```
2. **反应**: 
   ```
   Si-OH + Ca(OH)2 → C-S-H凝胶
   ```
3. **膨胀**: 
   - 碱-硅凝胶吸水膨胀
   - 膨胀压导致开裂

**宏观特征**: 
- **地图状裂纹(Map Cracking)**
- 白色凝胶渗出

- **来源**: Penn State + ASR paper
- **验证状态**: ✅ 交叉验证

### 5.3 ASR发生条件

| 条件 | 临界值 |
|------|--------|
| **碱含量** | Na2Oeq > 3.0 kg/m3 |
| **活性骨料** | 含活性SiO2 |
| **潮湿环境** | 相对湿度>80% |

**低碱水泥**: Na2Oeq < 0.6%

- **来源**: ASR论文 + 中国碱骨料反应规范
- **验证状态**: ✅ 交叉验证

### 5.4 ASR测试方法

| 方法 | 标准 | 判据 |
|------|------|------|
| 砂浆棒法 | ASTM C227 | 6个月膨胀<0.10% |
| 快速砂浆棒法 | ASTM C1260 | 14天膨胀<0.20% |
| 混凝土棱柱法 | ASTM C1293 | 1年膨胀<0.04% |

- **来源**: ASR paper + ASR diagnosis
- **验证状态**: ✅ 验证

### 5.5 ASR预防措施

| 措施 | 说明 | 最低掺量 |
|------|------|----------|
| 低碱水泥 | Na2Oeq<0.6% | - |
| 粉煤灰 | F类 | 30% |
| 矿渣 | GGBS | 50% |
| 硅灰 | SF | 10% |
| 锂盐 | LiNO3等 | 按推荐 |
| 非活性骨料 | - | - |

- **来源**: ASR diagnosis + Concrete Durability Guide
- **验证状态**: ✅ 交叉验证
