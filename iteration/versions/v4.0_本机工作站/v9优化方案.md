#### 3.2.3 上下文压缩策略（Compaction）

**触发条件**（满足任一即触发）：
1. 当前上下文超过 8K tokens
2. 单次对话超过 50 轮
3. 距离上次压缩超过 2 小时
4. 铉手动触发压缩

**压缩流程**：

```
1. 识别核心信息
   - 保留：铉的指令、关键决策、当前任务状态
   - 丢弃：重复确认、模板回复、无关闲聊

2. 生成摘要（由便宜模型执行）
   - 输入：原始对话
   - 输出：结构化摘要 + 关键决策点

3. 存档原始内容
   - 保存到 .memory/archive/
   - 文件名：{角色}_{日期}_{哈希}.md

4. 更新工作上下文
   - 清理历史对话
   - 注入摘要作为"回忆"
```

**Compaction脚本位置**：`D:\AI\XuanShu\scripts/compaction.py`

**压缩到粒度**：
- 最小单元：单条对话（一个user+assistant）
- 归档单元：整日对话
- 保留摘要：最近7天 + 里程碑事件

**优先级**：P0

#### 3.3.3 API限流保护

**现状**：CC+Codex同时打MiMo可能429
**方案**：轻量级令牌桶限流

**实现原理**：
- 每个API Endpoint维护一个令牌桶
- 桶容量 = 最大并发数
- 补充速度 = 速率限制 / 时间
- 请求先拿令牌，无令牌则等待

**轻量实现**：

```python
# D:\AI\XuanShu\scripts/rate_limiter.py
import time
import threading
from collections import defaultdict

class TokenBucket:
    def __init__(self, capacity: int, refill_rate: float):
        self.capacity = capacity
        self.tokens = capacity
        self.refill_rate = refill_rate  # 每秒补充令牌数
        self.last_refill = time.time()
        self.lock = threading.Lock()
    
    def acquire(self, timeout: float = 30) -> bool:
        """获取令牌，超时返回False"""
        deadline = time.time() + timeout
        while time.time() < deadline:
            with self.lock:
                self._refill()
                if self.tokens >= 1:
                    self.tokens -= 1
                    return True
            time.sleep(0.1)
        return False
    
    def _refill(self):
        now = time.time()
        elapsed = now - self.last_refill
        self.tokens = min(self.capacity, 
                         self.tokens + elapsed * self.refill_rate)
        self.last_refill = now

class RateLimiter:
    def __init__(self):
        self.buckets = defaultdict(lambda: TokenBucket(
            capacity=3,      # 最大并发3
            refill_rate=1    # 每秒补充1个令牌
        ))
    
    def limit(self, endpoint: str):
        """限流装饰器"""
        def decorator(func):
            def wrapper(*args, **kwargs):
                bucket = self.buckets[endpoint]
                if bucket.acquire(timeout=30):
                    return func(*args, **kwargs)
                else:
                    raise Exception(f"Rate limit exceeded for {endpoint}")
            return wrapper
        return decorator

# 全局实例
rate_limiter = RateLimiter()

# 使用示例
@rate_limiter.limit("mimo-api")
def call_mimo(prompt: str):
    # API调用逻辑
    pass
```

**限流配置**（config.yaml）：

```yaml
rate_limits:
  mimo_api:
    capacity: 3        # 最大并发
    rate: 1.0         # 每秒补充令牌
    timeout: 30       # 等待超时
    
  openclaw_api:
    capacity: 5
    rate: 2.0
    timeout: 30
```

"""
铉枢铁律执行器
从 D:\AI\XuanShu\.wiki/IRON_LAWS.md 读取规则并执行检查
"""

import re
import subprocess
from pathlib import Path
from typing import List, Tuple

class IronLawsEnforcer:
    def __init__(self):
        self.laws = self._load_laws()
    
    def _load_laws(self) -> dict:
        """从Wiki加载铁律"""
        laws_file = Path("D:\AI\XuanShu\.wiki/IRON_LAWS.md").expanduser()
        # 解析MD中的规则
        return {
            "must": [...],    # 必须执行的检查
            "forbidden": [...],  # 禁止的模式
        }
    
    def check_command(self, command: str) -> Tuple[bool, str]:
        """检查命令是否违反铁律"""
        
        # 检查危险路径
        dangerous_paths = [
            r"rm\s+-rf\s+/",           # 删除根目录
            r"rm\s+-rf\s+/System",     # 删除系统目录
            r"format\s+[A-Z]:",        # 格式化磁盘
        ]
        
        for pattern in dangerous_paths:
            if re.search(pattern, command, re.IGNORECASE):
                return False, f"禁止执行危险命令: {command}"
        
        # 检查文件删除
        if "rm " in command and not self._confirm_file_path(command):
            return False, "删除文件前必须确认路径"
        
        return True, "OK"
    
    def _confirm_file_path(self, command: str) -> bool:
        """确认命令中包含明确路径"""
        # 提取命令中的路径参数
        paths = re.findall(r"/[\w/.-]+", command)
        return len(paths) > 0  # 简单检查：是否有路径
    
    def enforce(self, action: str, context: dict) -> bool:
        """执行铁律检查，返回是否允许执行"""
        
        if action == "command":
            return self.check_command(context.get("command", ""))
        
        if action == "cross_role_command":
            # 检查是否绕过铉直接指挥
            return False, "禁止跨角色直接指挥"
        
        return True, "OK"

2. 危险命令直接拒绝，不进入LLM

#### 3.4.2 异常隔离

**现状**：单个任务失败会导致整个流程崩溃
**方案**：任务级隔离 + 失败重试

**实现**：

```python
# D:\AI\XuanShu\scripts/task_isolation.py
from enum import Enum
from typing import Callable, Any
import logging

class TaskStatus(Enum):
    PENDING = "pending"
    RUNNING = "running"
    SUCCESS = "success"
    FAILED = "failed"
    RETRYING = "retrying"

class IsolatedTask:
    def __init__(self, task_id: str, func: Callable, max_retries: int = 3):
        self.task_id = task_id
        self.func = func
        self.max_retries = max_retries
        self.status = TaskStatus.PENDING
        self.error = None
    
    def execute(self) -> Any:
        """隔离执行任务"""
        for attempt in range(self.max_retries):
            try:
                self.status = TaskStatus.RUNNING
                result = self.func()
                self.status = TaskStatus.SUCCESS
                return result
            except Exception as e:
                self.error = e
                self.status = TaskStatus.RETRYING
                logging.warning(f"Task {self.task_id} failed (attempt {attempt+1}): {e}")
                continue
        
        self.status = TaskStatus.FAILED
        logging.error(f"Task {self.task_id} failed after {self.max_retries} retries")
        return None  # 失败不影响其他任务
```

**解决的问题**：失败级联、错误蔓延
**参考**：Dagent异常隔离 + Gradientsys重试+重规划
**优先级**：P1
