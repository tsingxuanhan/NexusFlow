#### Phase 0: 修网络（前置依赖）

> 修好后才能进行后续所有步骤

1. 开机→趁有网那几秒→**设置→网络→代理→关掉手动代理**
2. **任务管理器→启动项→禁用所有VPN/代理类软件**（Clash/V2Ray/SSR等）
3. 打开PowerShell管理员，执行：
   ```
   ipconfig /flushdns
   netsh winsock reset
   netsh int ip reset
   ```
4. 重启，确认网络稳定

**验证**：`ping baidu.com` 和 `nslookup google.com` 都通才算修好。
