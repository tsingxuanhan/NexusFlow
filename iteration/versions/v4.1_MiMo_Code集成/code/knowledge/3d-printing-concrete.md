## 一、3D打印混凝土概述

### 1.1 技术特点

**优势**:
- 提高生产效率
- 降低人工需求
- 减少材料浪费

**挑战**:
- 可打印性、建造性、强度发展的平衡
- 水泥用量高（需控制流变性）

**来源**: Springer (2025)
**验证状态**: ✅ 论文验证

### 1.2 关键工艺参数

**流变学要求**:
- 可挤出性（Extrudability）
- 丝材形状保持性（Filament shape retention）
- 足够开放时间（Open time）

**力学要求**:
- 早期强度发展
- 层间结合
- 最终强度

**来源**: PMC (2025)
**验证状态**: ✅ 综述验证

## 二、低碳3D打印材料

### 2.1 地聚物3D打印

**特点**:
- 碳排放仅为OPC的1/5
- 早期强度发展快
- 非常适合快速施工

**来源**: 原创力文档 (2026)
**验证状态**: ⚠️ 待验证

### 2.2 纤维素纳米纤维+石灰石填料（Nature 2026）

**论文信息**:
- "Cellulose nanofibers and limestone filler enable high-performance, sustainable, and cost-efficient printable concrete"
- *Nature Communications*, 2026

**技术突破**:
- 0.3%纤维素纳米纤维 + 29%石灰石填料替代

**性能提升**:
| 参数 | 提升幅度 |
|------|----------|
| 静态屈服应力 | 1213% |
| 储能模量 | 255% |
| 临界应变 | 542% |

**效果**:
- 水泥用量减少40%
- 保持力学性能
- 大规模机器人3D打印验证

**来源**: Nature Communications (2026)
**验证状态**: ✅ DOI验证

### 2.3 锂渣+PVA纤维3D打印

**技术路线**:
- 锂渣（LS）作为SCMs
- PVA纤维增强

**锂渣特性**:
- 中值粒径38μm
- 比表面积342 m²/kg
- 无定形相31%
- 火山灰活性889 mg/g Ca(OH)₂

**来源**: Springer (2025)
**验证状态**: ✅ 论文验证

## 三、可打印性评价指标

### 3.1 关键参数

| 参数 | 说明 |
|------|------|
| 静态屈服应力 | 维持形状能力 |
| 动态屈服应力 | 挤出能力 |
| 触变指数 | 时间敏感性 |
| 开放时间 | 可操作窗口 |

### 3.2 SCMs对可打印性的影响

**火山灰活性比较**（mg Ca(OH)₂/g）:
- 硅灰(SF): 875
- 偏高岭土: 1050
- 粉煤灰(FA): 427
- 矿渣: 63

**影响规律**:
- 低中值粒径和高比表面积
- 丰富的无定形铝硅酸盐相
- 增强可打印性、建造性和强度发展

**来源**: Springer (2025)
**验证状态**: ✅ 论文验证

## 四、废旧材料在3D打印中的应用

### 4.1 生命周期评估（LCA）

**CO₂减排效果**:
- 粉煤灰、矿渣等固废替代：减排20-50%
- 再生细骨料：配合水泥减量可达48%
- 再生混凝土粉：可达62%

**来源**: PMC (2025)
**验证状态**: ✅ 综述验证

### 4.2 再生骨料应用

**技术要求**:
- 再生骨料应用比例>30%
- 破碎废弃混凝土分级处理
- 满足打印工艺骨料级配要求

**来源**: PMC (2025)
**验证状态**: ✅ 综述验证

### 4.3 导电与功能化混凝土

**功能类型**:
- 碳纳米管/石墨烯导电混凝土
- 电热转换与应力监测
