# 计算机科学核心概念

## 算法与数据结构
- **时间复杂度**: O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2^n)
- **排序算法**: 快排O(n log n)平均/归并O(n log n)稳定/堆排O(n log n)原地
- **图算法**: BFS(最短路径无权)/Dijkstra(带权)/A*(启发式)/拓扑排序(DAG)
- **动态规划**: 最优子结构+重叠子问题，自底向上填表
- **贪心算法**: 局部最优→全局最优，需证明贪心选择性质

## 操作系统
- **进程vs线程**: 进程=资源隔离单位，线程=调度单位，协程=用户态调度
- **虚拟内存**: 页表映射+TLB加速，缺页中断时换入
- **死锁**: 互斥/占有等待/非抢占/循环等待，预防/避免/检测
- **I/O模型**: 阻塞/非阻塞/I/O多路复用(epoll)/信号驱动/异步(AIO)

## 计算机网络
- **TCP三次握手**: SYN→SYN+ACK→ACK，确保双方收发能力
- **TCP四次挥手**: FIN→ACK→FIN→ACK，TIME_WAIT等待2MSL
- **HTTP/2**: 多路复用、头部压缩、服务器推送
- **HTTPS**: TLS握手→对称加密通信，RSA/ECDHE密钥交换

## 数据库
- **ACID**: 原子性/一致性/隔离性/持久性
- **索引**: B+树(范围查询)/哈希(精确查询)/全文(文本搜索)
- **事务隔离**: 读未提交→读已提交→可重复读→串行化
- **CAP定理**: 一致性/可用性/分区容错，最多同时满足两个

## 分布式系统
- **一致性算法**: Raft(领导者选举+日志复制)/Paxos
- **分布式ID**: 雪花算法(时间戳+机器ID+序列号)
- **负载均衡**: 轮询/加权/最少连接/一致性哈希
- **微服务**: 服务发现/熔断/限流/链路追踪
