## 五、ZIF-8 MOF阻锈剂

### 5.1 ZIF-8阻锈剂对钢筋砂浆的影响

**论文信息**:
- **标题**: 类沸石锌基咪唑酯金属有机框架阻锈剂在钢筋砂浆中的阻锈作用及机理
- **期刊**: 硅酸盐学报
- **年份**: 2025
- **作者**: (侯鹏坤相关论文)

**核心发现**:
- ZIF-8延长钢筋锈蚀诱发时间
- 0.8%掺量，180d阻锈效率62.1%
- 提高保护性产物含量，降低氯离子含量

- **验证状态**: ⚠️ 维普期刊论文
