# 数学与统计核心概念

## 线性代数
- **矩阵乘法**: (m×k)·(k×n)=(m×n)，O(n³)朴素算法
- **特征分解**: Av=λv，对称矩阵正交对角化
- **SVD**: A=UΣVᵀ，任意矩阵分解，降维/推荐/图像压缩
- **范数**: L1(稀疏性)/L2(平滑性)/L∞(最大值)
- **正定性**: xᵀAx>0对所有非零x，Hessian正定=局部极小

## 概率统计
- **贝叶斯定理**: P(A|B)=P(B|A)P(A)/P(B)，先验→后验
- **大数定律**: 样本均值→期望，n→∞时收敛
- **中心极限定理**: 独立同分布随机变量之和趋向正态分布
- **假设检验**: H₀/H₁，p值<α拒绝H₀，I型错误(弃真)/II型错误(取伪)
- **置信区间**: 95%CI=μ±1.96σ/√n

## 微积分
- **梯度**: 多元函数的偏导数向量，指向最速上升方向
- **链式法则**: ∂f/∂x=∂f/∂u·∂u/∂x，反向传播的数学基础
- **拉格朗日乘子**: 约束优化，∇f=λ∇g
- **泰勒展开**: f(x)=f(a)+f'(a)(x-a)+½f''(a)(x-a)²+...

## 信息论
- **熵**: H(X)=-Σp(x)log p(x)，不确定性度量
- **KL散度**: D_KL(P||Q)=ΣP(x)log(P(x)/Q(x))，非对称
- **交叉熵**: H(P,Q)=-ΣP(x)log Q(x)，分类损失函数
- **互信息**: I(X;Y)=H(X)-H(X|Y)，X与Y的信息关联

## 优化
- **SGD**: 随机梯度下降，小批量估计梯度
- **Adam**: 自适应学习率，动量+RMSProp
- **学习率调度**: Warmup→Cosine Decay/Step Decay
- **正则化**: L1(Lasso稀疏)/L2(Ridge平滑)/Dropout/BatchNorm
