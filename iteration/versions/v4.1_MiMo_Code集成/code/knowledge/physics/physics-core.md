# 物理学核心概念

## 力学
- **牛顿三定律**: 惯性/F=ma/作用力反作用力
- **能量守恒**: 动能+势能=常数，无外力做功
- **动量守恒**: 系统内力不改变总动量
- **拉格朗日力学**: L=T-V，广义坐标下的变分原理
- **刚体转动**: Iω=L，角动量守恒

## 热力学
- **第一定律**: ΔU=Q-W，能量守恒
- **第二定律**: 熵增原理，孤立系统熵只增不减
- **卡诺效率**: η=1-T_c/T_h，理想热机效率上限
- **麦克斯韦关系**: 热力学势之间的偏导关系
- **相变**: 一阶(潜热)/二阶(连续)，临界现象

## 电磁学
- **麦克斯韦方程组**: 四个方程统一电磁理论
- **库仑定律**: F=kq₁q₂/r²
- **法拉第定律**: 变化磁通产生感应电动势
- **安培定律**: 电流产生磁场

## 量子力学
- **薛定谔方程**: iℏ∂ψ/∂t=Ĥψ，波函数演化
- **不确定性原理**: ΔxΔp≥ℏ/2
- **量子隧穿**: 粒子穿越经典不可越势垒
- **自旋**: 内禀角动量，费米子半整数/玻色子整数

## 统计力学
- **玻尔兹曼分布**: P∝e^(-E/kT)
- **配分函数**: Z=Σe^(-Eᵢ/kT)，联系微观与宏观
- **费米-狄拉克**: 费米子分布，泡利不相容
- **玻色-爱因斯坦**: 玻色子分布，可占据同一态
