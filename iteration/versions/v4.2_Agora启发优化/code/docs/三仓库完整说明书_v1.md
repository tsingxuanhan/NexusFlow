## 3. 部署指南

### 3.1 本机部署（Windows 11 + RTX 3080 Ti 16GB）

#### 前置条件

| 依赖 | 最低版本 | 检查命令 |
|------|---------|---------|
| Python | 3.9+ | `python --version` |
| Node.js | 22+ | `node --version` |
| Git | 2.x | `git --version` |
| Docker Desktop | Latest | `docker --version` |
| Ollama | Latest | `ollama --version` |

#### 部署步骤

**步骤1: Clone仓库并配置**

```powershell
# 克隆核心框架
git clone https://github.com/tsingxuanhan/xuan-hub.git D:\AI\xuan-hub
cd D:\AI\xuan-hub

# 复制配置模板
copy config.example.py config.py

# 编辑config.py，填入真实环境变量
# DEEPSEEK_API_KEY=sk-your-key-here
# DEEPSEEK_ENDPOINT=http://127.0.0.1:8083/v1/chat/completions
```

**步骤2: 安装依赖**

```powershell
pip install -r requirements.txt
```

依赖包:
- requests>=2.28.0
- fastapi>=0.100.0
- uvicorn>=0.23.0
- aiohttp>=3.8.0
- numpy>=1.24.0
- pydantic>=2.0.0
- python-dotenv>=1.0.0

**步骤3: 启动Docker服务（可选）**

```powershell
# 方式A: 使用一键脚本（推荐）
.\scripts\start-local.ps1

# 方式B: 手动启动
docker compose -f docker\cloud-sim\docker-compose.yml up -d
```

**步骤4: 启动API代理**

```powershell
# 设置环境变量（可选，config.py中也已配置）
$env:DEEPSEEK_API_KEY="sk-your-key-here"
$env:DEEPSEEK_ENDPOINT="http://127.0.0.1:8083/v1/chat/completions"

# 启动三端口代理
python api-proxy/proxy.py

# 输出示例:
# 铉枢 API Proxy Layer
# ========================================
#   ✓ DeepSeek Gateway → https://api.deepseek.com (:8083, RPM=30)
#   ✓ MiMo 700M Gateway → https://token-plan-cn.xiaomimimo.com (:8084, RPM=20)
#   ✓ MiMo 200M Gateway → https://token-plan-cn.xiaomimimo.com (:8085, RPM=15)
```

**步骤5: 启动AgentOS**

```powershell
# HTTP模式（默认端口9090）
python agentos.py

# 或指定端口
python agentos.py --host 127.0.0.1 --port 9090

# stdio模式（用于其他Agent集成）
python agentos.py --mode stdio
```

**步骤6: 验证部署**

```powershell
# 运行快速验证脚本
python examples\quickstart.py

# 预期输出:
# Found 3 results
# Conflicts: 0
# Guardrail: block
# Quickstart done!
```

## 4. 使用指南

### 4.1 基础用法

#### 创建Agent

```python
from base_agent import BaseAgent

# 创建Agent
agent = BaseAgent(
    name="materials_assistant",
    system_prompt="You are a materials science AI assistant specializing in low-carbon cement."
)

# 设置模式
agent.set_mode("precise")  # creative/balanced/precise

# 调整质量旋钮
agent.set_dials(creativity=0.3, precision=0.9)
```

#### 对话与推理

```python
# 标准对话
response = agent.chat("What is SSC?")

# React推理模式
response = agent.react("Analyze the hydration mechanism of LC3")

# 自审模式
review = agent.review(response, criteria=["accuracy", "completeness"])
```

#### 向量记忆使用

```python
from vector_memory import EnhancedVectorMemory

# 创建记忆实例
evm = EnhancedVectorMemory()

# 设置Slot（上下文槽位）
evm.set_slot("research", "low-carbon cement", scope="user")

# 添加知识
evm.add("SSC main component is slag", importance=0.7, is_fact=True)
evm.add("Nano SiO2 can adjust SSC setting time", importance=0.6)

# 混合检索（NGramTFIDF + 向量 + RRF融合）
results = evm.hybrid_retrieve("What is SSC made of?", top_k=3)

# 冲突检测
conflicts = evm.get_conflicts()
```

#### Multi-Hop RAG

```python
from multi_hop_rag import MultiHopRAG

rag = MultiHopRAG(max_hops=3, top_k=5)
results = rag.query("How does LC3 improve durability?")
```

### 4.3 自主模式

```python
from autonomous import AutonomousGoalHandler

handler = AutonomousGoalHandler(
    max_decomposition_depth=3,
    max_execution_steps=20,
    confidence_threshold=0.6
)

# 6阶段处理
result = handler.process_goal(
    goal="Design a low-carbon concrete mix with 40% strength",
    domain="materials"
)
```

**6阶段处理器:**

| 阶段 | 功能 |
|------|------|
| 感知 | 理解目标，提取关键约束 |
| 分析 | 分解子目标，识别依赖 |
| 规划 | 生成执行计划，确定工具序列 |
| 执行 | 调用工具，更新状态 |
| 验证 | 检查结果是否符合目标 |
| 反思 | 总结经验，更新记忆 |

### 4.4 Agent协作

#### A2A协议

```python
from a2a_protocol import A2AProtocol

a2a = A2AProtocol(self_agent_id="miner")

# 广播消息
a2a.broadcast("Looking for: cement reactivity data")

# P2P直连
response = a2a.send_message("assayer", {"query": "validate this data"})

# 能力发现
capabilities = a2a.discover_capabilities()
```

#### Agent Handoff

```python
from handoff import AgentHandoff

handoff = AgentHandoff()

# 交接上下文
handoff.transfer(
    from_agent="miner",
    to_agent="assayer",
    context={
        "task": "data validation",
        "findings": [...],
        "confidence": 0.85
    }
)
```

| API代理连不上 | DEEPSEEK_API_KEY未设置 | `export DEEPSEEK_API_KEY="sk-xxx"` |
