## 一、概述与分类

### 1.1 定义

**辅助胶凝材料(SCMs)**: 
- 粉状矿物质材料
- 以硅、铝、钙等氧化物为主要成分
- 制备混凝土或砂浆时掺用
- 部分替代水泥熟料

**分类**:

| 类型 | 说明 | 代表材料 |
|------|------|----------|
| **活性SCMs** | 含活性SiO2/Al2O3，与碱反应生成胶凝产物 | 硅灰、矿渣、粉煤灰 |
| **非活性SCMs** | 无或少量活性组分 | 石灰石粉 |

- **来源**: 中国百科网 + FHWA SCMs Guide
- **验证状态**: ✅ 交叉验证

### 1.2 SCMs的作用方式

| 方式 | 机理 | 说明 |
|------|------|------|
| **填充效应** | 细颗粒填充大孔 | 立即改善孔结构 |
| **晶核效应** | 为水化产物提供成核位点 | 加速水化 |
| **火山灰反应** | SiO2+Al2O3+CH→C-S-H | 生成额外C-S-H |

- **来源**: NBMCW SCMs article
- **验证状态**: ✅ 验证

### 1.3 掺量参考

| SCM | 典型掺量范围 | 备注 |
|-----|--------------|------|
| GGBS | 30-70% | 可达80% |
| 粉煤灰(F类) | 15-30% | - |
| 粉煤灰(C类) | 20-35% | - |
| 硅灰 | 5-10% | 需配合减水剂 |
| 偏高岭土 | 10-20% | 高火山灰活性 |

- **来源**: Concrete Metric SCM Guide + FHWA
- **验证状态**: ✅ 交叉验证

## 二、粒化高炉矿渣(GGBS)

### 2.1 基本特性

**来源**: 高炉炼铁时，浮在铁水表面的熔融物经淬冷而成

**化学组成**:
| 成分 | 含量 |
|------|------|
| Al2O3 | 主要活性组分 |
| CaO | ~35-45% |
| SiO2 | ~30-35% |
| MgO | ~5-15% |

**活性指数分级**:
- S75: 7d≥75%, 28d≥95%
- S95: 7d≥95%, 28d≥115%
- S105: 7d≥105%, 28d≥115%

- **来源**: 中国百科网 + GB/T 18046
- **验证状态**: ✅ 交叉验证

### 2.2 水化机理

**潜在水硬活性**:
- 淬冷形成不稳定玻璃质结构
- 碱性环境下激活水化
- 主要水化产物: C-S-H

**水化反应**:
```
矿渣玻璃体 + Ca(OH)2 + 水 → C-S-H凝胶 + C-A-S-H凝胶
```

- **来源**: Concrete Metric + FHWA
- **验证状态**: ✅ 交叉验证

### 2.3 对混凝土性能的影响

| 性能 | 影响 |
|------|------|
| 和易性 | 改善黏聚性 |
| 水化热 | 降低(适合大体积) |
| 早期强度 | 可能降低 |
| 后期强度 | 显著提高 |
| 耐久性 | 改善，尤其抗氯盐 |
| 抗硫酸盐 | 良好 |

- **来源**: NBMCW + Concrete Metric
- **验证状态**: ✅ 交叉验证

### 2.4 胶凝效率系数

**公式**: 
```
k = SCM抗压强度贡献 / 水泥抗压强度贡献
```

| SCM | k值 |
|-----|-----|
| GGBS | ~0.9 |
| 粉煤灰(F类) | ~0.4 |
| 硅灰 | ~2.0 |

- **来源**: Concrete Metric SCM Guide
- **验证状态**: ✅ 验证

## 三、粉煤灰(Fly Ash)

### 3.1 基本分类

| 类型 | 来源 | CaO含量 | 火山灰活性 |
|------|------|---------|-----------|
| **F类** | 无烟煤/烟煤 | <10% | 高，需CH激发 |
| **C类** | 褐煤/亚烟煤 | 15-35% | 有一定水硬性 |

- **来源**: FHWA MCTC Guide + NBMCW
- **验证状态**: ✅ 交叉验证

### 3.2 物理特性

| 特性 | 说明 |
|------|------|
| 形态 | 空心或实心玻璃微珠 |
| 微珠含量 | 越多越好 |
| 烧失量 | 越低越好 |
| 细度 | 越细越好 |

**化学组成**:
- SiO2 + Al2O3: 通常>60%
- CaO: F类<10%, C类15-35%

- **来源**: FHWA + NBMCW
- **验证状态**: ✅ 交叉验证

### 3.3 火山灰反应机理

**反应方程式**:
```
SiO2 + Al2O3 + Ca(OH)2 + H2O → C-S-H + C-A-S-H
```

**反应阶段**:
1. **早期**: 形态效应改善和易性
2. **中期**: 填充效应
3. **后期**: 火山灰反应消耗CH

- **来源**: FHWA SCMs Guide + Concrete Metric
- **验证状态**: ✅ 交叉验证

### 3.4 对混凝土性能的影响

**优点**:
- 改善和易性(球形颗粒润滑)
- 节约水泥
- 降低水化热
- 提高后期强度
- 提高抗渗性
- 控制碱骨料反应(ASR)

**缺点**:
- 早期强度发展慢
- 需更长养护时间
- 可能增加碳化深度

- **来源**: FHWA + NBMCW
- **验证状态**: ✅ 交叉验证

## 四、硅灰(Silica Fume)

### 4.1 基本特性

**来源**: 冶炼硅铁合金或金属硅时，经烟道冷却收集的微细球形颗粒

**特性参数**:

| 参数 | 典型值 |
|------|--------|
| 粒径 | <1μm，平均0.1μm |
| 比表面积 | >15 m2/g |
| SiO2含量 | >85% |
| 形态 | 极细玻璃球状 |
| 结晶度 | 无定形(高活性) |

- **来源**: 中国百科网 + Concrete Metric
- **验证状态**: ✅ 交叉验证

### 4.2 作用机理

| 机理 | 说明 |
|------|------|
| **填充效应** | 填充水泥颗粒间微孔 |
| **火山灰反应** | 与CH反应生成C-S-H |
| **改善ITP** | 改善骨料-浆体界面 |

**反应方程式**:
```
SiO2(纳米) + Ca(OH)2 + H2O → C-S-H
```

- **来源**: Concrete Metric + NBMCW
- **验证状态**: ✅ 交叉验证

### 4.3 对性能的影响

**强度提升**:
- 抗压强度提高: 10-30%
- 火山灰反应生成额外C-S-H
- 细化孔结构

**耐久性改善**:
| 性能 | 改善程度 |
|------|----------|
| 抗渗性 | 显著提高 |
| 抗冻性 | 提高 |
| 抗化学侵蚀 | 提高 |
| 抗磨性 | 提高 |

**推荐掺量**: 5-10%（需配合高性能减水剂）

- **来源**: Concrete Metric + IRJMETS 2025
- **验证状态**: ✅ 交叉验证

## 五、偏高岭土(Metakaolin)

### 5.1 基本特性

**来源**: 高岭土在500-800°C煅烧脱羟基

**化学组成**:
- Al2O3: 35-45%
- SiO2: 45-55%

**特性**:
- 无定形结构(高火山灰活性)
- 比表面积大
- 反应活性高于粉煤灰

- **来源**: Concrete Metric + NBMCW
- **验证状态**: ✅ 交叉验证

### 5.2 作用机理

**火山灰反应**:
```
Al2O3 + SiO2 + Ca(OH)2 + H2O → C-A-S-H凝胶
```

**特点**:
- 反应速率快于粉煤灰
- 同时消耗CH和C-S-H
- 形成致密微观结构

- **来源**: Concrete Metric
- **验证状态**: ✅ 验证

### 5.3 对性能的影响

| 性能 | 影响 |
|------|------|
| 早期强度 | 提高 |
| 后期强度 | 显著提高 |
| 抗渗性 | 显著提高 |
| 抗ASR | 有效 |
| 颜色 | 可用于白色混凝土 |

**推荐掺量**: 10-20%

- **来源**: Concrete Metric + NBMCW
- **验证状态**: ✅ 交叉验证

## 七、其他SCMs

### 7.1 磷渣粉

**来源**: 电炉冶炼黄磷时的熔融物经淬冷

**特点**:
- 主要成分: SiO2 + CaO (>85%)
- 玻璃体含量: 85-90%
- 具有较高潜在活性

- **来源**: 中国百科网
- **验证状态**: ✅ 验证

### 7.2 稻壳灰(RHA)

**来源**: 稻谷加工副产物燃烧后收集

**特性**:
- SiO2含量: 85-95%
- 无定形或结晶
- 取决于燃烧温度

**效果**:
- 提高强度
- 改善耐久性
- 绿色环保

- **来源**: NBMCW + Concrete Metric
- **验证状态**: ✅ 交叉验证

### 7.3 石灰石粉

**定义**: 石灰岩粉碎加工，粒径<0.16mm

**作用**:
- 填充密实效应
- 改善和易性
- 提高强度(有限掺量)
- 降低绝热温升

**推荐掺量**: 需控制，过量可能影响耐久性

- **来源**: 中国百科网
- **验证状态**: ✅ 验证
