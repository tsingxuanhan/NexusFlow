## 技术要点
- **思想结晶**: 不是存原始数据，而是提炼推理过程中的思想精华
  - ThoughtDiamond 数据结构：核心思想 + 推理链 + 置信度 + 适用域
  - JSON持久化，最多100条结晶内存控制（适配3.8GB RAM）
- **抽象层级**: 知识分层管理
  - L1 事实层 → L2 推理层 → L3 策略层 → L4 元规则
  - 高层可约束低层，元规则指导策略选择
- **置信度门控**: 信息准入控制
  - 4维加权评分：逻辑一致性 + 事实校验 + 冗余检测 + 来源可信度
  - 决策：ACCEPT / REJECT / MERGE / NEEDS_REVIEW
- **与v4.2集成**: 结晶后自动写入 StructuredPatternMemory；MERGE时更新已有条目
- 纯增量设计：不修改任何v4.2文件
