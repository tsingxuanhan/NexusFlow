🧠 模型训练场

输出: 抗压强度预测

目标精度: R² > 0.85

输入: 分子图结构
架构: GCN×2 + GlobalPooling
输出: 分子性质预测
应用: C-S-H凝胶分子
任务类型: 分类/回归
LSTM (RNN)
输入: 配比 + 时间序列
架构: LSTM×2 + FC
输出: 强度发展曲线
应用: 养护龄期预测
目标精度: R² > 0.85
